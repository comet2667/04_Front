* jQuery란?

기존 자바스크립트 언어만으로 구현하기에는 복잡하고, 간소화하기 위해 만들어진 자바스크립트 기반의 라이브러리(유용한 기능 모음집)
즉, 자바스크립트 언어로 유용한 함수나 내용들이 이미 정의되어 있는 모음
복잡한 자바스크립트 구문을 보다 짧고 쉽게 개발 가능

=> 장점
1) DOM 요소와 관련된 스크립트를 쉽게 구현
2) ajax (비동기식 통신) 이벤트 처리를 폭 넓게 지원
3) jQuery와 관련된 확장형 플러그인, 오픈API 등을 지원(자료, 슬라이더, 캘린더, ...)

* jQuery 라이브러리 연결방법

head 태그 안에 script 태그로 외부 라이브러리 (jquery) 연결 - .js 파일 연결 (2가지)
1. jquery 라이브러리를 직접 다운로드 받아 경로를 지정(오프라인)
2. CDN 방식 (온라인) => 파일 다운로드 없이, url로 연결

* jQuery 구문 작성하기

jQuery 구문으로 DOM요소를 다루는 구문을 기본적으로 작성하기 때문에
HTML 문서 상, 요소들이 모두 만들어지고 나서 실행되어야 함 

        > 순수 자바스크립트(바닐라js)에서는 onload를 사용
        window.onload = function() {
            //모든 요소가 표시된 이후 실행할 코드
        }

        > jQuery에서는 아래와 같이 사용
        $(document).ready(function() {
            //실행 코드
        });

        $(function() {
            //실행 코드
        });

jQuery의 반복문

  1. for in 문법
  2. each 메소드

1) for in 문법

        //for
        for(초기식 ; 조건식 ; 증감식) {

        }

        //for in
        for(변수타입 변수명 in 배열명) {
            //변수명 => 인덱스
        }
        
        for(변수타입 변수명 in 객체명) {
            //변수명 => 속성명(key)
            //객체명[변수명] => 속성값(value);
        }

        //for of
        for(변수타입 변수명 of 배열명) {
            //변수명 => 배열 내 데이터
        }

2) each 메소드

        $.each(배열명, function(인덱스로 사용할 변수명-index, 해당데이터를 사용할 변수명-value){
            // index => 배열 인덱스 값
            // value => 배열 내 데이터 값
        });

        $(배열변수).each(function(index, value){
            // index => 배열 인덱스 값
            // value => 배열 내 데이터 값
        });
